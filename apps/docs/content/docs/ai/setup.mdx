---
title: Setup
description: Configure AI processing in settings and start indexing your files.
---

## 1. Open AI settings

Go to **Settings -> AI**.

## 2. Enable AI processing

Turn on **AI Processing**.

Then choose which features to enable:

- Embeddings
- OCR / text extraction
- Object detection

## 3. Configure processing

Set **Max concurrency** based on your available resources, then save.

## 4. Download models

In **AI Models**:

1. Choose a model size for each category.
2. Click **Download** for each category.
3. Wait for each category to show **Downloaded**.

## 5. Start indexing

In **Progress**, click **Start processing**.

Drivebase analyzes eligible files in the background.

## 6. Monitor progress

Track:

- Completion percentage
- Processed / eligible files
- Pending, running, failed, skipped, and completed counts

Progress updates are realtime.

## 7. If files fail

Use **Retry failed files** in the Progress section.

## 8. After settings changes

If you change model size or feature toggles:

- Download newly selected models if needed
- Re-run processing for best results

## 9. Optional cleanup

Use **Delete AI data** to clear indexed AI data and start fresh.
